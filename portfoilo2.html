ELECTRICITY GENERATION FROM UP AND DOWN MOTION

ELECTRICITY GENERATION FROM UP AND DOWN MOTION

Learn More

Electricity generation from up and down motion refers to the generation of electricity by the up and down motion of a see-saw. Nowadays, when we try to produce electricity, it causes some pollution such as noise pollution from hydro, radiation from nuclear, air pollution from thermal, etc. However, this prototype does not cause any pollution; it is completely eco-friendly. It works on the principle of electromagnetic induction and spur gear mechanism. Here, when we apply some force on either side, it generates up and down motion. I have placed a rod at the center of the see-saw which generates rotational energy, and this rotational energy is transferred to a dynamo using the spur gear mechanism. Finally, the dynamo converts it into electrical energy. With the help of this prototype, we can meet the demands of electricity, reduce park's electricity bill, recharge electric vehicles, etc.

AIRLINE RESERVATION SYSTEM

AIRLINE RESERVATION SYSTEM

Learn More

The "Airline Reservation System" portfolio project is a web application designed using HTML, CSS, and Bootstrap. Users can access the system to search for flights, view available seats, and make reservations. The interface features a user-friendly design created with HTML for structure, CSS for styling, and Bootstrap for responsiveness and layout consistency across devices. The application allows users to input their travel details, including departure and destination cities, date, and number of passengers. Upon submission, the system displays available flights along with their details and seat availability. Users can then select their preferred flight and proceed to make a reservation. The project demonstrates proficiency in front-end web development skills and showcases the ability to create a functional and visually appealing interface using fundamental web technologies.
